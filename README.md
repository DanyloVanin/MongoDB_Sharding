# MongoDB_Sharding

Sharded MongoDB configuration taken from [Bitnami Containers](https://github.com/bitnami/containers/tree/main/bitnami/mongodb-sharded)

## Running 

```sh
docker-compose -f docker-compose-multiple-shards.yml up --detach

```
